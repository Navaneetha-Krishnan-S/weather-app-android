# Weather App 🌦️

Welcome to the Weather App! This Android application provides real-time weather updates for any city you choose. 🌍

## Features ✨
- **Real-time Weather Updates** 🌡️
- **City Search** 🔍
- **Temperature in Celsius** 🌡️
- **Weather Descriptions** ☁️
